# Aeroleads Assignment

This repository contains three projects as part of the Aeroleads technical assignment:

1. **LinkedIn Scraper** - Automated LinkedIn profile data extraction
2. **Autodialer App** - Rails-based automated calling system with AI
3. **Blog Generator** - AI-powered blog content generation

## Project Structure

```
.
├── linkedin_scraper/     # LinkedIn profile scraping tool
├── autodialer_app/       # Ruby on Rails autodialer application
└── blog_generator/       # AI blog content generator
```

## Quick Start

Each folder contains its own README with specific setup instructions.

### 1. LinkedIn Scraper
Navigate to `linkedin_scraper/` for profile scraping functionality.

### 2. Autodialer App
Navigate to `autodialer_app/` for the Rails-based calling system.

### 3. Blog Generator
Navigate to `blog_generator/` for AI-powered blog generation.
